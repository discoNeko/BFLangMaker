BFLangMaker
- 任意の単語で brainf**k インタプリタを生成するウェッブサービス
STEP.1 単語を入力
brainf**k の命令を任意の単語に置き換える。
『 > 』 : ポインタを一つ進める。
『 < 』 : ポインタを一つ戻す。
『 . 』 : 現在ポインタが指している値を出力する。
『 + 』 : 現在ポインタが指している値を1増やす。
『 - 』 : 現在ポインタが指している値を1減らす。
『 [ 』 : ポインタの指している値が
0なら対応する『 ] 』の位置に進む。
『 ] 』 : ポインタの指している値が
0でないなら対応する『 [ 』の位置に戻る。
言語の名前を設定する。
STEP.2 ファイルを保存
Hello world のソースを確認する。
出力に満足したらダウンロードボタンをクリック。
html形式でファイルを保存する。
